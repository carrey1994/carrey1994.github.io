config with heroku 
@Configuration
public class DatabaseConfig {

    @Value("${spring.datasource.url}")
    private String dbUrl;

    @Bean
    public DataSource dataSource() {
        HikariConfig config = new HikariConfig();
        config.setJdbcUrl(dbUrl);
        return new HikariDataSource(config);
    }

}
postgreSQL DB create Server
Step1: open pgAdmin > Servers > Create > Server...
Step2: Gerneral tab > Name : give it a name your like (ex. Heroku)
Step3: Connection tab > fill up the forum by heroku Database Credentials (hint:check saving password is good)
Step4(Important): Advanced tab > give DB name to DB restriction
P.s. Forgiving this step, the database will result to some problem
