RUN touch log CMD ["/bin/bash", "-l", "-c", "./main > log 2>&1"] Docker Image製作 Terminal: docker build -t REGISTRY/IMAGE-NAME:TAG . -t 命名Image名稱和添加tag，並可push上的registry carrey1994/docker-play:release //Dockerhub上的空間 localhost:5000/docker-play:release //local registry 111.222.333.444:5000/docker-play:release //remote private registry docker push carrey1994/docker-play:release //將docker image 推到carrey1994的registry上 Docker Container運行 Docker run -d -p 8080:8080 --name play docker-play 將運行容器於背景並將8080 port forward到本機 Docker stop ContainerId or ContainerName 可將運行中的container停止 Docker start ContainerId or ContainerName 可將已停止的container重新運行 Docker exec -it CONTAINER-ID bash 可進入CONTAINER容器中以bash下達指令 Docker compose可以一次起多個docker container 例如可以下載https://github.com/deviantony/docker-elk.git以docker compose來快速運行啟動ELK。 Docker其他相關 docker save -o ~/docker-play.tar docker-play 可將Image製成.tar檔 docker load -i ~/docker-play.tar 可將.tar檔載進即可看到docker image docker tag docker-play NEW-IMAGE-NAME:NEW docker volume 此功能可將檔案進行mount 簡單運行redis container在Docker上 docker run -d -p 6379:6379 --name mredis redis //若docker images無redis則會自動搜尋redis image並將其pull下來。 docker exec -it mredis redis-cli
