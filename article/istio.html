Istio Note

 First, downlaod the Istio from  here. 
     
istioctl install --set profile=demo
cd istio-1.5.4/samples/helloworld
kubectl apply -f helloworld.yaml
kubectl apply -f helloworld-gateway.yaml
kubectl edit svc istio-ingressgateway -n istio-system //To get rid of pending status
kubectl port-forward svc/istio-ingressgateway LOCAL-PORT:80 -n istio-system
curl localhost:LOCAL-PORT/hello
It will show the version1 or version2. //Important
